SEO

发表于 2016-07-27   |  

SEO (Search Engine Optimization)是为了提升网页在搜索引擎自然搜索结果中的收录数量及排序位置而做的优化行为。

分为两类：

  • 白帽 SEO：改良和规范网站设计的作用，使之对搜索引擎和用户更加友好，并且网站也能从搜索引擎中获取更多合理的流量。
  • 黑帽 SEO：利用和放大搜索引擎政策缺陷来获得更多用户的访问量。这类行为大多是欺骗搜索引擎。（不推荐）

<!–more–>

####白帽 SEO：

  • 对网站标题、关键字、描述进行精心设计，反映网站定位，让搜索引擎明白网站是做什么的
  • 网站内容与关键字对应，增加关键字的密度
  • 在网站上合理设置 Robot.txt 文件
  • 生成针对搜索引擎友好的网站地图
  • 增加外部链接引用，到各个网站上宣传

以上都是网站内容上的 SEO，作为前端工程师，要从网站结构布局优化和网页代码优化上来进行SEO。

#####网站结构布局优化
合理规划站点结构，尽量让网站简单，开门见山。提倡扁平化结构。

要做到扁平化结构，注意以下几点：

  • 控制首页链接数量（不能没有，不能太多，对于中小型网站，首页链接控制在100个以内，链接性质可以包括页面导航、底部导航、锚文字链接等，放上的链接要建立在用户的良好体验和引导用户获取信息的基础上）
  • 扁平化的目录层次（尽量跳转三次就可以到达网站内任何一个内页）
  • 导航 SEO 优化 （一般分为主导航、父导航、分类导航。在每个网页上要加上面包屑导航，可以让用户了解当前所处的位置以及当前网页在整个网站中的位置，可以让用户快速了解网站组织形式）
  • 细节（网站布局方面，一般是：上面为导航；中间左边为面包屑导航和正文，中间右边为热门文章；下面为版权信息。 分页显示最好是：首页 1 2 3 4 5 下拉列表；有了下拉列表可以很快速的定位到任何一页）
  • 网站加载速度，控制页面大小，一般一个页面控制在 100k

#####网页代码优化：

  • <title..> 标题： 只强调重点即可，把重要的关键词放在前面，不要重复出现。每个页面的 title 不要设置相同的
  • <meta keywords> 关键词： 列举出几个重要关键词即可，不要过分堆砌。
  • <meta description> 网页描述：高度概括网页内容，不可太长，也不要过分堆砌关键字，每个页面的描述也要不同

对于 body 中的代码，强调语义化代码。
语义化代码： html 每个标签有特定的意义，语义化就是在适当的位置用适当的标签，以更好地让人和搜索引擎都一目了然。比如：

  • H1 - H6 标签多用于标题
  • UL 标签多用于无序列表
  • OL 标签多用于有序列表
  • DL 标签多用于定义数据列表
  • em, strong 表示强调

针对一些常用的标签，分析怎么 SEO

  • <a…> 标签，要加上 title 用于对链接的说明， 对外部的链接要加上 rel=“nofollow”， 防止小蜘蛛爬出去
  • <h1..>标签 正文标题要用 h1标签，小蜘蛛认为 h1 标签的标题最重要（如果觉得字体太大，可以用 css 修饰） 副标题用 h2。其他不重要的标题不要滥用 h 标签。
  • <p…>标签与<br>标签 注意：<br>标签只用于文本内的换行，如果是其他元素的换行，应该通过调整 css 样式，比如 margin、padding 等。
  • <table..> 标签内 使用 <caption…>定义表格名称
  • <img…> 标签应使用 alt 说明
  • <strong…> <em…> 与 <b…> <i…>标签
  • 巧妙使用 css 布局，把重要内容 HTML 代码放在最前面
  • 重要内容不要用 JS 输出
  • 尽少使用 iframe 框架
  • 谨慎使用 display: none;
  • 不断精简代码

JS 控制 float 属性

由于 float 是 JavaScript 的保留字，因此不能直接使用 obj.style.float 。其正确的使用方法是：在 IE 下用 styleFloat， Firefox， chrome， Safari 用 cssFloat，可用 in 运算符去检测 style 是否包含此属性。

LTFunction.setFloatStyle = function(obj,style) { 
    var sty=obj.style; 
    if('cssFloat' in sty){ 
        obj.style.cssFloat=style; 
    }else if('styleFloat' in sty){ 
        obj.style.styleFloat=style; 
    }else{ 
        throw 'set float style:'+style+'error.'; 
    } 
}

高级 css 选择器

发表于 2016-05-29   |   分类于 JavaScript   |  

1. 属性选择器

有6个不同类型的属性选择器：

  • [att=value] 该属性有指定的确切的值。
  • [att~=value] 该属性的值必须是一系列用空格隔开的多个值，(比如，class=”title featured home”)，而且这些值中的一个必须是指定的值”value”。
  • [att|=value] 属性的值就是“value”或者以“value”开始并立即跟上一个“-”字符，也就是“value-”。(比如lang=”zh-cn”)
  • [att^=value] 该属性的值以指定值开始。
  • [att$=value] 该属性的值包含指定的值(而无论其位置)。
  • [att*=value] 该属性的值以指定的值结束
