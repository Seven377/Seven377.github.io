高级 css 选择器

发表于 2016-05-29   |   分类于 JavaScript   |  

1. 属性选择器

有6个不同类型的属性选择器：

  • [att=value] 该属性有指定的确切的值。
  • [att~=value] 该属性的值必须是一系列用空格隔开的多个值，(比如，class=”title featured home”)，而且这些值中的一个必须是指定的值”value”。
  • [att|=value] 属性的值就是“value”或者以“value”开始并立即跟上一个“-”字符，也就是“value-”。(比如lang=”zh-cn”)
  • [att^=value] 该属性的值以指定值开始。
  • [att$=value] 该属性的值包含指定的值(而无论其位置)。
  • [att*=value] 该属性的值以指定的值结束

发表于 2016-03-29   |   分类于 Web   |  

IP 协议

规定Internet数据传输的基本单元和格式的协议，网络中传输的信息都要遵守IP协议指定的信息传输的大小和格式，定义了数据的路由选择。

TCP协议

TCP协议提供可靠的数据传输服务的规则，IP协议可以进行数据包的分割和组装，但是通过它不能清楚的了解数据包是否顺利的发送给目标计算机，TCP协议将数据包成功发送给目标计算机时，会要求发送一个确认，如果在某个时间内没有收到确认，TCP将重新发送数据包，另外，在传输的过程中，如果接收到无序、丢失以及被破坏的数据包，TCP还负责恢复。
